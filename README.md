## How it works
This is a fake online REST API for testing and prototyping. Powered by [json-server](https://github.com/typicode/json-server).

## Use base URL `https://my-json-server.typicode.com/RicharC293/fake_doctors` and add endpoints to get the data you need. For example, to get all doctors, use `https://my-json-server.typicode.com/RicharC293/fake_doctors/doctors`.

## Endpoints

Base URL:

`https://my-json-server.typicode.com/RicharC293/fake_doctors`

Available resources from `db.json`:

- `GET /posts`
- `GET /posts/:id`
- `GET /comments`
- `GET /comments/:id`
- `GET /profile`
- `GET /doctors`
- `GET /doctors/:id`
- `GET /services`
- `GET /services/:id`
- `GET /products`
- `GET /products/:id`
